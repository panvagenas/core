# glob( string $dir, string $pattern, boolean $case_insensitive = FALSE, null|integer $x_flags = NULL, null|integer $flags = NULL, string $type = websharks_core_v000000_dev\fw_constants::file_type )

Glob files.

Glob files.

Parameters

$dir
string

$dir Glob directory.

$pattern
string

$pattern A glob pattern string.

$case_insensitive
boolean

$case_insensitive Optional. This defaults to a FALSE value. If TRUE, this enables the websharks_core_v000000_dev\fw_constants::glob_casefold flag (as described below).

• If TRUE, we force character classes on both $dir & $pattern (i.e. [aA][bB]) to test for caSe variations. See also: websharks_core_v000000_dev\strings::fnm_case() for further details about how this works.

$x_flags
null|integer

$x_flags Optional. Defaults to a NULL value. The defaults are: websharks_core_v000000_dev\fw_constants::glob_period, websharks_core_v000000_dev\fw_constants::glob_brace; unless $flags are passed in explicitly. Regardless, if you'd like to use additional flags, pass this parameter with a non-NULL value.

• You may NOT use the websharks_core_v000000_dev\fw_constants::glob_nocheck flag with this method. It is NOT supported here. Attempts to use the websharks_core_v000000_dev\fw_constants::glob_nocheck flag will result in an exception.

$flags
null|integer

$flags The defaults are recommended; but specific flags can be passed in if you prefer. The difference between $x_flags and $flags; is that $flags will override all defaults; whereas $x_flags will simply add additional flags to the existing defaults.

• PHP's own glob() function will NOT (by default) find hidden dot . files using wildcards. However, WE use the websharks_core_v000000_dev\fw_constants::glob_period, websharks_core_v000000_dev\fw_constants::glob_brace flags as follows: glob('/{,.}*', GLOB_BRACE). This functionality is enabled automatically; when/if $flags includes websharks_core_v000000_dev\fw_constants::glob_period, websharks_core_v000000_dev\fw_constants::glob_brace; and whenever your $pattern contains /* combinations. Instances of /* are converted into /{,.}* glob brace checks.

  In short, we find all hidden dot `.` files automatically (this is the default behavior).
  To disable this default behavior, pass ``$flags`` w/o the websharks_core_v000000_dev\fw_constants::glob_period flag.

• You may NOT use the websharks_core_v000000_dev\fw_constants::glob_nocheck flag with this method. It is NOT supported here. Attempts to use the websharks_core_v000000_dev\fw_constants::glob_nocheck flag will result in an exception.

$type
string

$type Optional. This defaults to websharks_core_v000000_dev\fw_constants::any_type. If you want to ONLY glob directories, use: websharks_core_v000000_dev\fw_constants::dir_type. If you want to ONLY glob files, use: websharks_core_v000000_dev\fw_constants::file_type.

Returns

array
