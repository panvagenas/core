$include_scalars_resources
boolean

$include_scalars_resources Optional. Defaults to FALSE. By default, we do NOT objectify scalars and/or resources. Typecasting scalars and/or resources to objects, counterintuitively results in an object with a single scalar property. While this might be desirable in some cases, it is FALSE by default.
