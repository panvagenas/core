public boolean $updates_options FALSE
#

Defaults to FALSE. Does this menu page update options? When TRUE, each menu page is wrapped with a form tag that calls ©options.®update. In addition, $this->option_fields will be populated, for easy access to a ©form_fields instance. In addition, each menu page will have a Save All Options button.

Defaults to FALSE. Does this menu page update options? When TRUE, each menu page is wrapped with a form tag that calls ©options.®update. In addition, $this->option_fields will be populated, for easy access to a ©form_fields instance. In addition, each menu page will have a Save All Options button.

Extenders

Can easily be overridden by class extenders.


Note

This comes in handy, when a menu page is dedicated to updating options. Making it possible for a site owner to update all options (i.e. from all panels), in one shot. The Save All Options button at the bottom will facilitate this.
