Command line execution processor.

Command line execution processor.

Parameters

$cmd_args
string

$cmd_args Command and arguments.

$stdin
string

$stdin Optional standard input to this command. This will be written to the input pipe.

$return_array
boolean

$return_array Return full array? Defaults to FALSE. If TRUE, this method will return an array with all connection details. The default behavior of this function is to simply return a string that contains any output received from the command line routine.

$return_errors
boolean

$return_errors If an error occurs, return an errors object instead of a string or array? Defaults to FALSE. If TRUE, upon an error occurring, an errors object will be returned instead of a string or array.

$cwd
string

$cwd The initial working dir for the command. This must be an absolute directory path. Defaults to an empty string. If needed, this parameter can by bypassed with an empty string. Actually, this defaults to a NULL value, but that's handled internally. To bypass this argument, use an empty string.

$env
array

$env An array with the environment variables for the command that will be run, or an empty array to use the same environment as the current PHP process. Defaults to an empty array. Actually, this defaults to a NULL value, but that's handled internally. To bypass this argument, use an empty array.

$other
array

$other Allows you to specify additional options. Defaults to an empty array. Actually, this defaults to a NULL value, but that's handled internally. To bypass this argument, use an empty array.

Returns

string|array|wsc_v000000_dev\errors

The default behavior of this function is to simply return a string that contains any output received from the command line routine. The return value of this function will depend very much on the command that was actually called upon. ~ And, of course, also based on the argument values: $return_array, $return_errors.

Returns

string|array

The output from Git; always a string. However, this will throw an exception if Git returns a non-zero status. If $return_array is TRUE, we simply return the full array of connection details, regardless of what Git returns.


Throws

If invalid types are passed through arguments list.


wsc_v000000_dev\exception
