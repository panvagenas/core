Throws

wsc_v000000_dev\exception

Note

This is an EXTREMELY COMPLEX routine that should NOT be modified without serious consideration.


Given this complex/lengthy routine, the result of ANY call to this method is cached statically. An MD5 checksum is used to determine if we have ALREADY validated the $fields prior. If we have, there is NO need to validate them again.


Security-note

Callers should NEVER process disabled form fields (for security purposes). It's always possible for disabled fields to be submitted by malicious users (not common, but possible). This method does NOT validate disabled fields (we do NOT expect these to be submitted). Any routines that process form submissions should be sure NOT to process these.
