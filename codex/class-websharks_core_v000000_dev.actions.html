Generates a link/URL that performs a dynamic ©class.®method action call.

Parameters

string

$call A dynamic ©class.®method action call that we need to make.

$type
string

$type Call type. One of these class constants (e.g. types):

• Public — websharks_core_v000000_dev\fw_constants::public_type This type of call is less secure. It does NOT require a logged-in user. Public call actions MUST be registered by class methods containing a ® in their name. Public call actions do NOT include a timestamp; and they do NOT expire.

• Protected — websharks_core_v000000_dev\fw_constants::protected_type This type of call is less secure. It does NOT require a logged-in user. Protected call actions MUST be registered by class methods containing a ® in their name. Protected call actions can include a custom expiration time (expires after 7 days, by default).

• Private — websharks_core_v000000_dev\fw_constants::private_type This type always requires a logged-in user matching the $verifier. Private call actions MUST be registered by class methods containing a ® in their name. Private call actions can include a custom expiration time (expires after 24 hours, by default).

$args
array

$args Optional. An array of arguments we're passing to $call.

$expires_after
integer

$expires_after Optional. Time (in seconds) the underlying call verifier should last for.

Some additional notes regarding the $expires_after parameter:

  • Public — websharks_core_v000000_dev\fw_constants::public_type
  Public `call` actions do NOT expire (ever). The string verifier will not even include an expiration time.
  Therefore, this argument is ignored when ``$type`` is websharks_core_v000000_dev\fw_constants::public_type.

  • Protected — websharks_core_v000000_dev\fw_constants::protected_type
  For protected `call` actions, this will default to `604800` (7 days).

  • Private — websharks_core_v000000_dev\fw_constants::private_type
  For private `call` actions, this will default to `86400` (24 hours).
$base
string

$base Optional. Defaults to WordPress® home URL. A base URL to use instead of the default value.

$use_action_key
boolean

$use_action_key Optional. Defaults to a TRUE value. Links are compressed w/ the use of an action key. If this is FALSE, we'll embed the action array into the link itself. However, this can produce MUCH longer/uglier links. If this is FALSE, the $action_key_expires_after parameter is ignored completely (e.g. irrelevant).

$action_key_expires_after
integer

$action_key_expires_after Optional. The time (in seconds) a link w/ an action key should last for. Defaults to the computed $expires_after value. Set this to override the default value. Set to -1 for no link expiration time. Note: it is usually NOT necessary to pass this value in, because it's synchronized with $expires_after.
